CSS (Cascading Style Sheets)

CSS es el lenguaje utilizado para describir la presentación de un documento HTML. Fue desarrollado por Håkon Wium Lie y Bert Bos y permite a los desarrolladores estilizar páginas web con colores, fuentes, diseño y más.

Objetivo y Función Principal

El objetivo principal de CSS es separar la presentación del contenido. Permite a los desarrolladores aplicar estilos a los elementos HTML, definiendo cómo se verán en un navegador. CSS puede controlar el diseño del sitio en diferentes dispositivos y tamaños de pantalla.

Características

  • Selectores: Seleccionan los elementos HTML que se van a estilizar.

  • Propiedades y valores: Definen qué aspecto tendrá un elemento (como color, margen, padding, etc.).

  • Cascada y Herencia: Los estilos se aplican en un orden específico y pueden heredarse de elementos padre a hijo.

  • Media queries: Permiten aplicar estilos específicos para diferentes tamaños de pantalla o dispositivos.

Css

body {

font-family: Arial, sans-serif;

}

h1 {

color: blue;

}

p {

font-size: 16px
line-height: 1.5;

}

a {

color: red;
text-decoration: none;

}
